Bootsmacs proposes you a rules of writing things and file separation to aviod chaos, but has been constructed to have in mind full control over the style properties and markup. It combines various trending technologies and approaches together, providing everything what is needed to deliver code that is scalable and easy to maintain, does not matter if it is a web app, huge CMS implementation or a small website.



Features


Pure CSS components

All typical jQuery use cases for basic interactions replaced with pure CSS techinques. All components are very basic and boilerplate.

Written in Sass

With power of variables, mixins and nesting inside CSS and clean syntax.

SMACSS and BEM

Separating files according so SMACCS methodology and using BEM naming convention makes your styles scallable and easy to maintains.

Flexbox grid

The most flexible grid system based on flex property.

Built in form reset

Fully hackable and configurable styles for the forms.

Built in helpers

Helper classes and usefull mixins to minimize the effort. Some of them you may know from Bootstrap

Gulp

< automation tool onboard to compile and preview your work in browser on file save.

Styleguide rendering

With KSS docummenting syntax you can render styleguide for your interface components.
