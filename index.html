Hackable in nature


Fronthack brings you simple, fully hackable structure to avoid mess known from Bootstrap. It proposes you a rules of writing and separating code, while giving you full control over the style properties and HTML markup. No !default variables, no overrides in Sass files.

Everything what you need


Combine together trending technologies and approaches. Everything what you need, does not matter if it is a web app, huge CMS implementation or a small website, but what exactly is inside?



SMACSS and BEM

Separating files according to SMACSS methodology and using BEM naming convention makes your styles scallable and easy to maintain.

Web components library

Package of web components that are very basic and boilerplate to do not reinvent the wheel.

Pure CSS

All typical jQuery use cases for basic interactions (except carousel) has become replaced with pure CSS techinques.

Written in Sass

Sasswith power of variables, mixins, nesting and clean syntax.

Flexbox grid

The most flexible Flexbox Grid system based on flex property.

Minimalistic form framework

Fully operational built in framework to style the forms. Cointains validation states.

Gulp

An automation tool onboard to compile and preview your work in browser on file save.

Built in helpers

Helper classes and usefull mixins to minimize the effort and keep the DRY rule.

OWL Carousel

Universal OWL Carousel 2 plugin implemented as one of the components.

CSS reset

Contains Normalize.css to set default appearance on all browsers.

Autoprefixer

Forget about writing vendor prefixes for CSS3 properties using Autoprefixer.

Mustache

Simple, oldschool templating engine to separate common header and footer from main content of HTML prototypes.

KSS Styleguide

With KSS docummenting syntax you can render the styleguide for your web components.
