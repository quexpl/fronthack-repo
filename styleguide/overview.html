Huge change - Bootsmacss is now standalone framework! We resigned from depending on Bootstrap! As a results file strucute become more clean, and SASS structure is more consistent. You can still use the Bootstrap version, but it is not supported anymore.

Bootsmacss

Bootsmacss is a fusion of technologies that will raise your workflow to new level. Make your markup and styles scalable and easy to maintain, does not matter if you are working on web app, huge CMS implementation or small website.

View Bootsmacss styleguide demo

Technologies included

  • Flexbox Grid - a grid system based on the flex display property.
  • FormHack - Minimalist form reset
  • SMACSS - modular stylesheet approach that splits everything into components
  • SASS - keeps your stylesheets's syntax clean and tidy
  • BEM - stylish structure for each SASS component
  • GULP - to automate various tasks and compile your work on file save
  • Mustache - to render static html prototypes as a result of graphic project's implementation that can be shown to client, tested and implemented later to CMS, or deployed to production server
  • KSS Styleguide - to render styleguide for your components easly
  • Autoprefixer - adds vendor prefixes to your CSS3 rules
  • Additional usefull stuff - Extra components, mixins, helper classes, etc. Some of them you may know from Bootstrap
