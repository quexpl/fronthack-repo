Getting started

1. You need Gulp installed, which is a Node.js package.

Using Mac? Do that with homebrew

brew update brew install node npm install --global gulp

Using Ubuntu?

sudo add-apt-repository ppa:chris-lea/node.js sudo apt-get update sudo apt-get install nodejs npm install --global gulp

2. Download or clone

Use download link or paste that at a Terminal prompt:

git clone --depth=1 https://github.com/frontcraft/fronthack.git cd fronthack rm -rf .git

3. Install project dependencies

While being in fronthack folder, run this in terminal:

npm install

Now you should learn How to work with Fronthack.
