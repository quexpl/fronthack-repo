counter Counter

Widget whith date and circular diagram how much time is left. Based on example http://codepen.io/rendro/pen/qazCG. Informations are shown horizontally or vertically to not take up unnecessary space on mobile devices depending on width of wrapping element. See example below how it works inside a bootstrap collumn (marked with dashed border). Resize browser window to see the differences.

86
days left 17-Jun-2015
.counter--fixed-big
Force counter always to be displayed in big version, on all resolutions.
86
days left 17-Jun-2015 
<div class="row"><div class="col-md-3"><div style="border: 1px dashed lightgray">
    <!-- Component markup -->
    <div class="counter [modifier class]" id="counter-0">
      <div data-percent="15.412182671082" class="counter__pie counter-pie-processed">
        <span class="counter__left">86</span>
        <canvas height="120" width="120"></canvas>
      </div>
      <span class="counter__label">days left</span>
      <span class="counter__caption">17-Jun-2015</span>
    </div>
    <script src="../../js/components/counter/easy-pie-chart.js"></script>
    <script src="../../js/components/counter/counter-demo.js"></script>
    <!-- /Component markup -->
</div></div></div>
