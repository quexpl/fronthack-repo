input-inner-addon Input inner addon

Icon or any another content to place inside the form input. It works similiar like Bootstrap Input groups, but appears inside the input, not next to it.

<div class="input-inner-addon__wrapper">
  <input type="text" class="form-control" placeholder="Search">
  <span class="input-inner-addon">
    <span class="icon icon--search"></span>
  </span>
</div>
